内网穿透工具tunnel.mobi使用教程

发表于 2017-02-14   |   分类于 webService   |  

该工具用于将内网ip转化为公网ip

通过该工具局域网之外的人也能够访问到你本地的搭建的web页面
目前市面上流行的内网穿透工具大概有以下几种
  • 花生壳（要钱，没用过）
  • localtunnel（nodejs模块，使用超级简单，但是巨tm慢，直接pass）
  • ngrok（同样慢的不行，pass）
  • tunnel.mobi（这个好，速度快，配置简单）

下面主要介绍使用方式

  1. 进入tunnel.mobi的github主页
    链接

  2. 下载对应的版本文件

  3. 解压到任意文件夹

  4. 下面开始具体操作流程

    1. 修改 frps.ini 文件，配置一个名为 web 的 http 反向代理，设置 http 访问端口为 8080，绑定自定义域名,www.*.com

    2. 启动 frps

    3. 修改 frpc.ini 文件，设置 frps 所在的服务器的 IP 为 x.x.x.x，local_port 为本地机器上 web 服务对应的端口：ip地址为本地局域网ip地址

    4. 启动 frpc：

    5. 将 www.*.com 的域名 A 记录解析到 IP x.x.x.x（就是刚刚配置的那个本机ip），如果服务器已经有对应的域名，也可以将 CNAME 记录解析到服务器原先的域名。

    6. 通过浏览器访问 http://www.*****.com:8080 即可访问到处于内网机器上的 web 服务。
